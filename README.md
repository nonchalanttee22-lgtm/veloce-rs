# ⚡ veloce-rs

> Ultra-low latency payment streaming on Stellar.

`veloce-rs` is a Soroban-native protocol designed for high-frequency value transfer. By leveraging the speed of the Stellar Network and the safety of Rust, Veloce enables per-second capital distribution with minimal overhead.

## 🛠 Tech
- **Core**: Rust (Soroban SDK)
- **Network**: Stellar Testnet/Futurenet
- **Safety**: Formal verification-ready logic
